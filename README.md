# ガーベラ

IRIAMライバー向けの配信支援Webツール（β版）。

- **公開URL（GitHub Pages）**: https://riorio513.github.io/gerbera/
- ログイン・会員登録・クラウド同期はなし（β版のため）。データはすべてブラウザ内（localStorage）に保存されます。
- 「今日の企画」を選ぶと、その企画に合ったツールだけが表示されます。画面下部の共通ツール（電卓・pt変換・タイマー・ストップウォッチ・リスナーメモ）はどの画面からでも開けます。

## 更新の流れ

ファイルを編集 → コミット → push すると、GitHub Pages が自動で更新されます。
